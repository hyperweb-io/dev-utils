## Development

### Setup

1. Clone the repository:

```bash
git clone https://github.com/hyperweb-io/dev-utils.git
```

2. Install dependencies:

```bash
cd dev-utils
pnpm install
pnpm build
```

3. Test the package of interest:

```bash
cd packages/<packagename>
pnpm test:watch
```
